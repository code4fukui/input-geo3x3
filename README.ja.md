# input-geo3x3

Geo3x3フォーマットの地理座標を入力できる入力フォームです。

## デモ
https://code4fukui.github.io/input-geo3x3/

## 機能
- Geo3x3フォーマットの座標入力
- 緯度経度の表示
- Geo3x3レベルの表示

## 必要環境
特に必要ありません。

## 使い方
HTMLに`<input-geo3x3>`要素を配置するだけで使用できます。

```html
<input-geo3x3 id=inpgeo3x3></input-geo3x3>
```

JavaScript から値を取得する際は、`inpgeo3x3.value`で取得できます。
